# javaweb-practice

자바 웹 개발 연습 프로젝트입니다.  
STS(Spring Tool Suite) 기반으로 JSP, Servlet, MVC 패턴 등을 학습합니다.

---

## 📁 프로젝트 개요

- **프로젝트 이름**: javaweb-practice
- **개발 목적**: 자바 웹 기초 이해 및 연습
- **사용 툴**: Eclipse STS, Tomcat, Git
- **기술 스택**:
  - Java 17
  - JSP / Servlet
  - HTML / CSS / JavaScript (기본 정적 자원)
  - JDBC (예정)
  - MVC 패턴

---

## 🚀 주요 학습 목표

- 웹 서버(Tomcat) 구동 및 프로젝트 배포
- JSP/Servlet 동작 원리 이해
- 폼 요청 처리(GET/POST)
- MVC 패턴으로 구조 분리
- 간단한 게시판, 로그인 구현 (예정)
